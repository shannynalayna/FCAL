Detailed Description

The Parser class is used to parse a Token list generated by the scanner into an AST that represents the syntax of the productions in the FCAL language. Most of the parse functions are split up by production type (Program, Stmts, Stmt, Decl, and Expr) and use sequential token matching and the classes from ast.h to build the AST and make sure it is grammatically correct. The code that we changed in iteration 3 involved updating the code to produce the AST using the newly created ast.h classes.
