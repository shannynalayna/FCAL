Detailed Description

The Scanner class is used to scan an FCAL program (inputted as a character string) for tokens and send them to a parser. The Scan function is the main component of this class; it does all of the regex matching to determine the next token in the program and creates a Token object list to be returned. This function uses an array of regex_t regular expressions (which correspond to FCAL language tokens) for token matching.
