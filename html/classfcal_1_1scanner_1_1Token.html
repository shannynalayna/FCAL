Detailed Description

The Token class maintains a singly linked list of Token objects. Each Token has a terminal of type TokenType (a previously defined enum that is used to identify the type of a token), a string lexeme that contains the actual text that was scanned for the token, and a pointer to the next Token in the list. Each of these parameters has a get and set funcition.
